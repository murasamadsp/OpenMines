# OpenMines roadmap

План восстановления Mines 3.
Клиент: Unity 2019.4.10f1 (Windows через Wine/GPTK).
Сервер: Rust + SQLite + mmap, TCP-binary.

---

## Фаза 1: MVP (подключение и мир) ✅ DONE

- [x] TCP tokio на `8090`
- [x] Протокол `[i32 len][u8 type][2B event][payload]`
- [x] Каналы: U UTF-8, B binary/HB, J JSON
- [x] Handshake ST → AU → PI/PO
- [x] NO_AUTH + GUI-регистрация (ник/пароль)
- [x] Token auth: MD5(hash + `session_id`)
- [x] World через `mmap` (cells/road/durability), 3 слоя
- [x] Chunk 32x32, видимость 5x5
- [x] Движение с дистанцией и коллизиями
- [x] Broadcast позиции в HB bot
- [x] SQLite: players/chats/chat_messages/buildings
- [x] Сохранение на disconnect
- [x] `cells.json`: 126 типов клеток

## Фаза 2: Копание, стройка, экономика ✅ DONE

- [x] Копание Xdig: урон, разрушение, FX broadcast
- [x] Кристаллы: CellType 71-75 (X), 107-112 (обычные)
- [x] Множители X-кристаллов x2-x4
- [x] Корзина B: 6 типов, лимиты
- [x] Стройка Xbld: G/R/O/V блоки
- [x] Апгрейд: GreenBlock → YellowBlock → RedBlock
- [x] Дороги, опоры, военные блоки
- [x] Автокопание TADG
- [x] Валуны при копании
- [x] Лечение Xhea: -1 red, +10 HP
- [x] Инвентарь IN/INVN/INUS/INCL
- [x] Геопаки: предметы 10-16, 34, 42-43, 46
- [x] 8 базовых скиллов + SK при логине
- [x] Смерть: потеря кристаллов, респавн, FX
- [x] Миграция БД: inventory, skills

## Фаза 3: Здания (packs) ✅ DONE

Здание = pack на карте, несколько клеток, GUI при входе.

- [x] **Pack-система**: buildings в БД, load on start, HB 'O'
- [x] **Resp** (1): респ-локация игрока, размер 6x3
- [x] **Teleport** (0): телепорт между точками, окно выбора, 2x1
- [x] **Up** (2): апгрейд скиллов за деньги, 2x4
- [x] **Market** (3): покупка/продажа кристаллов, 3x3
- [~] **Gun** (26): заряд есть, клан-атака по врагам — см. Фазу 5
- [x] **Gate** (27): клановые ворота, блокировка чужих
- [x] **Storage** (29): личное хранение, 2x1
- [x] **Crafter** (24): крафт из кристаллов, 2x2
- [x] **Spot** (32): нейтральная точка, без спецлогики
- [x] **GU-пакет**: окно входа на клетку здания (`check_pack_at_position`)
- [x] **Доступ**: владелец / клан / публичный (`clan_id` проверки)
